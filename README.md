# learn-graphql

> 本教材服务器基于`express`,`express-graphql`,`graphql`.

## 开始

1. 克隆库:

```shell
git clone https://github.com/mafeifan/learn-graphql
```

2. 安装依赖:

```shell
npm install
```

3. 运行各个例子:

```shell
npm src/1.command-hello.js
...
npm src/2.express-graphql.js
```

打开浏览器输入`http://localhost:3000/graphql`
